Vertical Geothermal / CO2 profile - Custom

This filter generates geothermal and CO2 vertical profiles for a custom user specified area (bounding box).
The minimum number of vertices for the bounding box is 4; i.e. a rectangular bounding box (additional vertices can be specified). For a rectangular box seperately enter the 4 latitude and 4 longitude coordinates seperated by a comma. For example for a bounding box with vertices of (30, -101), (32,-101), (32,-98) and (30,-98);
