Horizontal Geothermal / CO2 profile - Custom

This filter generates geothermal and CO2 horizontal profiles for a custom user specified area (bounding box).
The minimum number of vertices for the bounding box is 4; i.e. a rectangular bounding box (additional vertices can be specified). As far as I am aware there is no maximum number of vertices. For a rectangular box seperately enter the 4 latitude and 4 longitude coordinates seperated by a comma.

For example for a rectangular bounding box with vertices of (30, -101), (32,-101), (32,-98.5) and (30,-98.5);
    For latitude enter: 30,32,32,30
    For longitude enter: -101,-101,-98.5,-98.5
