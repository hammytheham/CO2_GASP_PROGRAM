This service allows users to run the R mixed modelling simulation to determine the local dolomite solubility constant (see Robertson et al. 2022). The user has the option to either upload a CSV file (download a correctly formatted sample here which uses a tab seperator) or use the default PWGD database (the results of which are found here). Uploading a CSV file results in a merge of the dataframes prior to running the mixed model; this enables the user to calculate a local dolomite solubility constant with an advantage of being able to use the sizeable USGS database as the 'reference data' which increases accuracy and relatibility.
